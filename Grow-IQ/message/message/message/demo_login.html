API Testing

Test the API endpoints directly using these curl commands. Perfect for developers who want to integrate with our platform.

Login API Examples

# Login with Alice curl -X POST "http://localhost:9000/api/v1/users/login" \ -H "Content-Type: application/json" \ -d '{ "email": "alice@example.com", "password": "securepassword123" }' # Login with Bob curl -X POST "http://localhost:9000/api/v1/users/login" \ -H "Content-Type: application/json" \ -d '{ "email": "bob@example.com", "password": "securepassword456" }'

Connection API Examples

# Send connection request curl -X POST "http://localhost:9000/api/v1/connections" \ -H "Content-Type: application/json" \ -H "Authorization: Bearer YOUR_JWT_TOKEN" \ -d '{"receiver_id": 2}' # Get connections curl -X GET "http://localhost:9000/api/v1/connections" \ -H "Authorization: Bearer YOUR_JWT_TOKEN"
